INSTALAÇÃO


Instalação no Windows:

A instalação do Go no Windows é similar a de outros programas (next, next, next, finish). Siga os passos a seguir para realizar a instalação:

1) Acesse a página oficial do Go para realizar o download do instalador.
2) Navegue até a pasta onde o instalador foi baixado.
3) Clique com o botão direito no instalador e selecione “Executar como Administrador”.
4) Com o instalador aberto, certifique-se de que o local de instalação é adequado e siga as instruções na tela.
5) Por fim, clique em “Install” e prossiga com o processo padrão de instalação.

Para verificar se a instalação foi bem-sucedida, abra o Prompt de Comando ou o Power Shell e digite o comando abaixo:

go version


Instalação no Linux:

Para instalar o Go no Linux, siga os passos abaixo:

1) Baixe o arquivo .tar.gz do site oficial do Go para a versão Linux.
2) Extraia o arquivo usando o comando:
sudo tar -C /usr/local -xzf go1.x.x.linux-amd64.tar.gz

3) Adicione o caminho do Go ao PATH do sistema, adicionando as linhas abaixo ao final do arquivo ~/.profile ou ~/.bashrc:
export PATH=$PATH:/usr/local/go/bin

4) Recarregue o perfil com o comando:
source ~/.profile

Para verificar a instalação, execute:
go version


Instalação no macOS:

A instalação do Go no macOS segue um processo padrão:

1) Baixe o instalador .pkg para macOS do site oficial do Go.
2) Dê um duplo clique no arquivo e siga as instruções de instalação na tela.

Para verificar a instalação, abra o Terminal e execute:
go version


Vantagens

Golang é Simples e Eficiente

Golang é uma linguagem minimalista e fácil de aprender, ideal para novos programadores e para desenvolvedores experientes que buscam produtividade.

Performance Elevada

Como linguagem compilada, o Go gera código de máquina diretamente, proporcionando uma performance superior em comparação a linguagens interpretadas.

Gerenciamento de Memória Eficiente

O Go possui um coletor de lixo (garbage collector) altamente eficiente, oferecendo bom desempenho em gerenciamento de memória.

Paralelismo Nativo

Golang possui suporte nativo a goroutines, facilitando a criação de programas com paralelismo e escalabilidade.


Desvantagens

Biblioteca Padrão Limitada

A biblioteca padrão do Go não é tão rica quanto a de outras linguagens, o que pode exigir o uso de bibliotecas de terceiros.

Falta de Generics

Embora o suporte a generics tenha sido introduzido recentemente, ele ainda está em evolução, limitando certas operações para usuários que vêm de linguagens como Java.


Aplicações

  • Desenvolvimento de sistemas distribuídos e em nuvem
  • Desenvolvimento de ferramentas de linha de comando

  • Desenvolvimento de sistemas de rede e aplicações de backend

Conceitos Básicos

Declaração de Variáveis

Em Go, variáveis são tipadas estaticamente. Você pode declarar variáveis usando a palavra-chave var ou o operador curto :=.


var idade int = 25
nome := "João"


Estruturas de Controle

O Go oferece estruturas de controle, como if, for e switch, para controlar o fluxo do programa.


if idade >= 18 {
fmt.Println("Maior de idade")
} else {
fmt.Println("Menor de idade")
}


Laços de Repetição

Em Go, o laço for é o único loop disponível, mas ele cobre diversos casos de repetição.


for i := 0; i < 5; i++ {
fmt.Println(i)
}


Funções

Funções são definidas com a palavra-chave func.


func saudacao(nome string) string {
return "Olá, " + nome
}
fmt.Println(saudacao("Maria"))


Comentários

Comentários em Go são precedidos por // para uma linha ou /* ... */ para múltiplas linhas.

Estruturas de Dados: Arrays, Slices e Maps

O Go possui arrays de tamanho fixo e slices de tamanho dinâmico.


nomes := []string{"João", "Maria", "José"}
idade := map[string]int{"João": 30, "Maria": 25}


Goroutines e Channels

Goroutines são a forma de Go para concorrência leve e canais são utilizados para comunicação entre goroutines.


go func() {
fmt.Println("Hello from goroutine")
}()
ch := make(chan int)

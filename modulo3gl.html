BUSCA E ORDENAÇÃO


BUSCA

O algoritmo de busca é amplamente utilizado em Go e muitas outras linguagens de programação para localizar elementos em uma coleção de dados. Dependendo da organização dos dados e da necessidade de otimização, diferentes tipos de algoritmos de busca podem ser utilizados, como a busca linear e a busca binária.


Busca Linear


A busca linear é um método simples de pesquisa, em que percorremos todos os elementos de uma lista ou array um a um até encontrar o elemento desejado ou atingir o fim da coleção. É útil em casos onde os dados não estão ordenados.

Em Go, um exemplo de busca linear:



            package main
            import "fmt"
            
            func buscaLinear(arr []int, x int) bool {
                for _, valor := range arr {
                    if valor == x {
                        return true
                    }
                }
                return false
            }
            
            func main() {
                lista := []int{10, 23, 42, 4, 15}
                fmt.Println("Número 23 encontrado?", buscaLinear(lista, 23))
            }

Busca Binária

A busca binária é mais eficiente que a busca linear, mas requer uma lista ordenada. O algoritmo divide a lista ao meio repetidamente até encontrar o elemento desejado ou determinar que ele não está presente.

Em Go, podemos implementar a busca binária assim:



            package main
            import "fmt"
            
            func buscaBinaria(arr []int, x int) bool {
                esquerda, direita := 0, len(arr)-1
                for esquerda <= direita {
                    meio := (esquerda + direita) / 2
                    if arr[meio] == x {
                        return true
                    }
                    if arr[meio] < x {
                        esquerda = meio + 1
                    } else {
                        direita = meio - 1
                    }
                }
                return false
            }
            
            func main() {
                lista := []int{2, 3, 5, 7, 11, 13}
                fmt.Println("Número 7 encontrado?", buscaBinaria(lista, 7))
            }

ORDENAÇÃO


O algoritmo de ordenação reorganiza os elementos de uma lista ou array em uma ordem específica, geralmente crescente ou decrescente. Aqui, exploramos três métodos comuns: ordenação por bolha, seleção e inserção.


Ordenação por Bolha

A ordenação por bolha compara elementos adjacentes e os troca se estiverem na ordem incorreta. Esse processo é repetido até que todos os elementos estejam ordenados.

Exemplo em Go:



            package main
            import "fmt"
            
            func ordenacaoBolha(arr []int) {
                n := len(arr)
                for i := 0; i < n-1; i++ {
                    for j := 0; j < n-i-1; j++ {
                        if arr[j] > arr[j+1] {
                            arr[j], arr[j+1] = arr[j+1], arr[j]
                        }
                    }
                }
            }
            
            func main() {
                lista := []int{5, 3, 8, 4}
                ordenacaoBolha(lista)
                fmt.Println("Lista ordenada:", lista)
            }


Ordenação por Seleção


A ordenação por seleção busca o menor (ou maior) elemento e o posiciona corretamente na lista. O processo é repetido para os elementos restantes até que a lista esteja ordenada.

Exemplo em Go:



            package main
            import "fmt"
            
            func ordenacaoSelecao(arr []int) {
                n := len(arr)
                for i := 0; i < n-1; i++ {
                    menor := i
                    for j := i + 1; j < n; j++ {
                        if arr[j] < arr[menor] {
                            menor = j
                        }
                    }
                    arr[i], arr[menor] = arr[menor], arr[i]
                }
            }
            
            func main() {
                lista := []int{5, 3, 8, 4}
                ordenacaoSelecao(lista)
                fmt.Println("Lista ordenada:", lista)
            }


Ordenação por Inserção


A ordenação por inserção constrói a lista ordenada um elemento de cada vez, inserindo cada novo elemento na posição correta em relação aos já ordenados.

Exemplo em Go:



            package main
            import "fmt"
            
            func ordenacaoInsercao(arr []int) {
                for i := 1; i < len(arr); i++ {
                    chave := arr[i]
                    j := i - 1
                    for j >= 0 && arr[j] > chave {
                        arr[j+1] = arr[j]
                        j = j - 1
                    }
                    arr[j+1] = chave
                }
            }
            
            func main() {
                lista := []int{5, 3, 8, 4}
                ordenacaoInsercao(lista)
                fmt.Println("Lista ordenada:", lista)
            }

Esses três algoritmos são básicos, mas fornecem um entendimento fundamental dos métodos de ordenação. Para listas maiores, há algoritmos mais eficientes, como quicksort e mergesort.

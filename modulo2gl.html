PROGRAMAS BÁSICOS

Fatorial

O fatorial é uma operação matemática usada no estudo de análise combinatória. Um fatorial é representado por um número seguido de um símbolo de exclamação (!), como exemplo: Quando usamos 5! estamos dizendo que queremos o fatorial de 5. No código, criamos uma função que calcula o fatorial utilizando um loop ou recursão, acumulando o produto dos números.



                package main
                import "fmt"
                
                func fatorial(n int) int {
                    if n == 0 {
                        return 1
                    }
                    resultado := 1
                    for i := 1; i <= n; i++ {
                        resultado *= i
                    }
                    return resultado
                }
                
                func main() {
                    fmt.Println("Fatorial de 5:", fatorial(5))
                }

Número primo

Um número natural é primo se ele possui apenas dois divisores positivos e distintos. Ou seja, um número natural é primo se ele é maior que 1 e é divisível apenas por si próprio e por 1. Por exemplo: o número 2. Ele só é divisível por ele mesmo e por 1.



                package main
                import "fmt"
                
                func ehPrimo(n int) bool {
                    if n <= 1 {
                        return false
                    }
                    for i := 2; i*i <= n; i++ {
                        if n%i == 0 {
                            return false
                        }
                    }
                    return true
                }
                
                func main() {
                    fmt.Println("7 é primo?", ehPrimo(7))
                }

Fibonacci

A sequência de Fibonacci é uma sequência numérica em que cada termo, a partir do terceiro, é a soma dos dois anteriores. Em Go, essa sequência pode ser calculada usando uma função iterativa ou recursiva, armazenando os resultados para otimizar o desempenho.



                package main
                import "fmt"
                
                func fibonacci(n int) int {
                    if n <= 1 {
                        return n
                    }
                    return fibonacci(n-1) + fibonacci(n-2)
                }
                
                func main() {
                    fmt.Println("Fibonacci de 6:", fibonacci(6))
                }

Euler

O método de Euler é um procedimento numérico de primeira ordem para resolver equações diferenciais ordinárias com um valor inicial. O número de Euler, uma constante fundamental em matemática, pode ser aproximado com somatórios em Go, onde a série infinita é truncada conforme necessário.



                package main
                import "fmt"
                
                func eulerAprox(precisao int) float64 {
                    resultado := 1.0
                    fatorial := 1
                    for i := 1; i <= precisao; i++ {
                        fatorial *= i
                        resultado += 1.0 / float64(fatorial)
                    }
                    return resultado
                }
                
                func main() {
                    fmt.Println("Aproximação de Euler:", eulerAprox(10))
                }

Vetor

Em Go, um vetor (array) é uma sequência de elementos com tamanho fixo. Para uma sequência dinâmica, utilizamos um slice. Slices são flexíveis e permitem operações como adicionar elementos dinamicamente, o que facilita a manipulação de grandes quantidades de dados.



                package main
                import "fmt"
                
                func main() {
                    // Definindo um array fixo
                    var array [5]int = [5]int{1, 2, 3, 4, 5}
                
                    // Definindo um slice dinâmico
                    slice := []int{1, 2, 3, 4, 5}
                
                    // Adicionando elementos no slice
                    slice = append(slice, 6)
                
                    fmt.Println("Array:", array)
                    fmt.Println("Slice:", slice)
                }

Matriz

Em Go, podemos representar uma matriz como um slice de slices. Cada slice representa uma linha da matriz, e seus elementos são acessados por índices duplos, como `mat[i][j]`. Isso permite armazenar e manipular dados bidimensionais, como tabelas de valores.



                package main
                import "fmt"
                
                func main() {
                    matriz := [][]int{
                        {1, 2, 3},
                        {4, 5, 6},
                        {7, 8, 9},
                    }
                
                    // Exibindo a matriz
                    for _, linha := range matriz {
                        fmt.Println(linha)
                    }
                }

Arquivo

- Criação

Para abrir e manipular arquivos em Go, usamos o pacote `os`. A função `os.Create()` cria um arquivo para escrita. Já `os.OpenFile()` permite especificar modos como leitura e gravação com permissões adicionais.

Para escrever no arquivo, utilizamos `Write()` ou `WriteString()`, e é importante fechar o arquivo após o uso com `Close()`.



                package main
                import (
                    "fmt"
                    "os"
                )
                
                func main() {
                    // Criando um arquivo
                    arquivo, err := os.Create("exemplo.txt")
                    if err != nil {
                        fmt.Println("Erro ao criar arquivo:", err)
                        return
                    }
                    defer arquivo.Close()
                
                    // Escrevendo no arquivo
                    _, err = arquivo.WriteString("Hello, Golang!")
                    if err != nil {
                        fmt.Println("Erro ao escrever no arquivo:", err)
                    }
                }

- Leitura de Arquivo

Para ler um arquivo em Go, utilizamos a função `os.Open()` e lemos o conteúdo usando funções como `Read()` ou `bufio.NewScanner()` para leitura linha a linha. Lembrando de fechar o arquivo com `Close()` após terminar a leitura.



                package main
                import (
                    "fmt"
                    "os"
                    "bufio"
                )
                
                func main() {
                    arquivo, err := os.Open("exemplo.txt")
                    if err != nil {
                        fmt.Println("Erro ao abrir arquivo:", err)
                        return
                    }
                    defer arquivo.Close()
                
                    scanner := bufio.NewScanner(arquivo)
                    for scanner.Scan() {
                        fmt.Println(scanner.Text())
                    }
                }

- Excluir Arquivo

Para excluir um arquivo, utilizamos a função `os.Remove()`, passando o nome do arquivo que queremos excluir. Caso o arquivo não exista, será retornado um erro.



                package main
                import (
                    "fmt"
                    "os"
                    "bufio"
                )
                
                func main() {
                    arquivo, err := os.Open("exemplo.txt")
                    if err != nil {
                        fmt.Println("Erro ao abrir arquivo:", err)
                        return
                    }
                    defer arquivo.Close()
                
                    scanner := bufio.NewScanner(arquivo)
                    for scanner.Scan() {
                        fmt.Println(scanner.Text())
                    }
                }
